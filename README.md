# GROWW Weekly Pulse

Automated weekly analysis of Play Store reviews using Groq LLM (Llama 3). Transforms raw feedback into high-impact product insights.

## 🚀 Features
- **Refined Scraping**: Fetches high-quality, English-only reviews.
- **AI Core**: Themes, quotes, and action ideas generated by Groq (Llama 3 70B).
- **Premium Dashboard**: GROWW-themed Streamlit UI for data visualization.
- **Automated Pulse**: Weekly reports dispatched via SMTP with CSV attachments.
- **Scheduled Pipeline**: Pre-configured for Windows Task Scheduler.

## 🛠️ Local Setup

### 1. Prerequisite
- Python 3.8+
- [Groq API Key](https://console.groq.com/)

### 2. Installation
Clone the repository and install dependencies:
```bash
git clone https://github.com/rushirp3942-netizen/GROWW-Weekly-Pulse.git
cd GROWW-Weekly-Pulse
pip install -r requirements.txt
```

### 3. Environment Configuration
Create a `.env` file in the root directory:
```env
GROQ_API_KEY=your_groq_api_key

# Email/SMTP Settings
SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587
SENDER_EMAIL=your_email@gmail.com
SENDER_PASSWORD=your_app_password
RECIPIENT_EMAIL=stakeholder_email@example.com
```
*Note: If using Gmail, use a [Google App Password](https://myaccount.google.com/apppasswords).*

## 📈 Running the Project

### Dashboard (User Interface)
```bash
streamlit run streamlit_app.py
```

### Manual Pipeline Execution
```bash
python Phase6/scheduler.py
```

### Automation (Windows)
Follow instructions in `walkthrough.md` to set up **Task Scheduler** pointing to `Phase6/scheduler.py`.

## 📁 Project Structure
